# Python small program for generating ASCII code table
## Dependencies
```
pip install pathlib
```
## How to Use
Run the generation program
```
python main.py
```
Then you can use the file
```
ASCII_gbk.txt
ASCII_utf8.txt
```
The underscore (_) is followed by the code!

Just open the `.txt` file and you're ready to use it!
